Admin User Management

- In Admin Management you can view all admin user accounts

  • You can add new admin user
  • You can edit current admin user account info
  • You can delete an admin user account


Customer Account Management

- In Customer Management you can view all customer accounts

  • You can add new customer account
  • You can edit current customer account info and shipping address
  • You can delete a customer account
  • You can help a customer to clear their shopping cart


Product Management

- In Products you can view all the products

  • You can add new product to the sotre
  • You can edit current product's info - product name, inventory, price...
  • You can delete a product
  • Note the list only show limited information, click on More in the table of
    each product to see more information of the product


Page/Store Layout Editor

- In Page Editor you can change the layout of the store page

  • You can change the background color and text color in Page Style
  • You can change logo in Change logo
  • You can change contact page in Edit Contact
  • You can change the policy pages
